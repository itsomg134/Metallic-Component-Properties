# Metallic Component Properties

A sleek, futuristic UI dashboard that visualizes material properties of industrial components with a **metallic, high-tech aesthetic**. Built with pure HTML & CSS — no JavaScript required.

![metallic-preview](https://via.placeholder.com/800x400/1a1e24/b0c4e0?text=Metallic+Component+Properties)

---

## Features

- **Metallic Design Language** – Gradients, shadows, and textures that mimic brushed steel, titanium, and copper.
- **6 Interactive Component Cards** – Each card represents a different alloy or metal (Steel, Titanium, Copper, Aluminum, Stainless Steel, Inconel).
- **At-a-Glance Properties** – Display key metrics like tensile strength, density, conductivity, and heat treatment.
- **Fully Responsive** – Adapts gracefully to any screen size, from desktop to mobile.
- **Pure CSS** – No JavaScript or external libraries (except Font Awesome for icons). Fast and lightweight.
- **Hover Effects** – Cards lift and glow on hover, revealing subtle metallic reflections.

## 🖥️ How to Use

1. **Clone or download** this repository.
2. Open `index.html` in any modern web browser.
3. Explore the component cards — hover to see interactive effects.

No build tools, servers, or dependencies required. Just open and go.

---

## 🎨 Customization

- **Add a new component** – Duplicate a `.metal-card` block inside `.component-grid` and update the values.
- **Change colors** – Modify the CSS variables or gradient stops in the `.metal-card` and `.metallic-wrapper` selectors.
- **Replace icons** – Swap the `<i class="...">` classes with any [Font Awesome](https://fontawesome.com/icons) icon.
